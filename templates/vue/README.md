# __APP_NAME__

An AD4M application scaffolded with [`create-ad4m-app`](https://github.com/coasys/create-ad4m-app).

## Stack

Vue 3 + Vite 8 + Tailwind v4 + TypeScript 5.6 + Playwright + Husky.

See [`templates/default/README.md`](../default/README.md) for the per-file walkthrough — this Vue variant uses `ref` / `onMounted` in place of Solid's signals.

## Quick start

```bash
pnpm install
pnpm dev
```

Requires a running AD4M executor.

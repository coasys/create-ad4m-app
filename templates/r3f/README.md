# __APP_NAME__

An AD4M application scaffolded with [`create-ad4m-app`](https://github.com/coasys/create-ad4m-app).

## Stack

React Three Fiber + Vite 8 + Tailwind v4 + TypeScript 5.6 + Playwright + Husky.

The 3D scene reflects live AD4M state: each note in the Scratch Perspective spawns a floating sphere. Hover for emerald highlight. The orbit autorotates.

## Quick start

```bash
pnpm install
pnpm dev
```

Requires a running AD4M executor.

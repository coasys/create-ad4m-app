# __APP_NAME__

An AD4M application scaffolded with [`create-ad4m-app`](https://github.com/coasys/create-ad4m-app).

## Stack

React 19 + Vite 8 + Tailwind v4 + TypeScript 5.6 + Playwright + Husky.

See [`templates/default/README.md`](../default/README.md) in the create-ad4m-app repo for the full per-file walkthrough — this React variant uses the same patterns with `useState`/`useEffect` in place of Solid's signals.

## Quick start

```bash
pnpm install
pnpm dev
```

Requires a running AD4M executor.
